ExplorerXP is a very fast, small, compact and innovative FREEWARE (for non - commercial use) file manager for Windows 2000/XP. It allows you to view, edit, copy, move, rename and delete files and folders within a drag-and-drop interface.
 

Tabbed interface - all your windows (open folders) will appear inside a single ExplorerXP application. The tab bar provides a quick access to all open windows

Easy access to My Computer, Recycle Bin & My Documents

Displays folder size information

Displays real size of compressed files & folders

Multy rename tool - allows to rename multiple files in one step

Merge / Split Tools

Advanced copy/move

Unicode support
 

Clean - recursively removes files that match given list of extentions or wildcards.
