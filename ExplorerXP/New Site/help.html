 Keyboard Shortcuts with ExplorerXP
Copy. CTRL+C
Cut. CTRL+X
Paste. CTRL+V
Delete. DELETE
Delete selected item permanently without placing the item in the Recycle Bin. SHIFT+DELETE
Copy selected item. CTRL while dragging an item
Rename selected item. F2
Select more than one file or folder. SHIFT with the arrow keys
Select all. CTRL+A
Quit ExplorerXP. ALT+F4
Close the active folder. CTRL+F4
Refresh the active window. F5
Move forward through tabs. CTRL+TAB
Move backward through tabs. CTRL+SHIFT+TAB
Display the bottom of the active window. END
Display the top of the active window. HOME
Open a folder one level up. BACKSPACE
View properties for the selected item (s). ALT+ENTER
Display the Address bar list. F4
Go to the Address bar. Alt+D
Display the shortcut menu for the selected item.
